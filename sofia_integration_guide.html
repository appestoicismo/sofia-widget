🚀 Método 1: Integração Simples (Recomendado)

Para sua página de vendas atual - apenas 3 linhas de código!

📁 1. Upload dos Arquivos

Faça upload destes arquivos para seu servidor/Netlify:

📂 Sua pasta do site/ ├── sofia_widget_integrado.js ├── Sofia_IA.png └── sua-pagina-de-vendas.html

📝 2. Adicione na sua Página

Cole este código antes do </body> da sua página de vendas:

<!-- Sofia Widget --> <script src="sofia_widget_integrado.js"></script> <script> SofiaWidget.init({ primaryColor: '#667eea', welcomeMessage: 'Olá! Sou a Sofia. Como posso te ajudar com o AppEstoicismo?', avatarUrl: 'Sofia_IA.png' }); </script>
✅ Pronto! A Sofia aparecerá automaticamente na sua página!

🔄 Método 2: GitHub + Netlify (Seu Workflow)

Usando seu workflow atual com deploy automático

📁 1. Estrutura no GitHub

📂 seu-repositorio/ ├── assets/ │ ├── sofia_widget_integrado.js │ └── Sofia_IA.png ├── index.html (sua página de vendas) └── README.md

📝 2. Na sua página de vendas

<!-- No final do <body> --> <script src="assets/sofia_widget_integrado.js"></script> <script> SofiaWidget.init(SofiaWidget.presets.estoic); </script>

🚀 3. Deploy Automático

git add . git commit -m "Adicionar Sofia Widget" git push origin main

Netlify fará deploy automaticamente! 🎉

☁️ Método 3: CDN (Mais Rápido)

Para máxima performance - carregamento global

📝 Cole na sua página:

<!-- Sofia Widget via CDN --> <script src="https://cdn.jsdelivr.net/gh/seuusuario/sofia-widget@main/sofia_widget_integrado.js"></script> <script> SofiaWidget.init({ primaryColor: '#667eea', avatarUrl: 'https://cdn.jsdelivr.net/gh/seuusuario/sofia-widget@main/Sofia_IA.png' }); </script>
💡 Vantagem: Carregamento mais rápido e cache global!

⚙️ Configurações Avançadas

🎨 Personalização Visual

SofiaWidget.init({ primaryColor: '#667eea', secondaryColor: '#764ba2', position: 'bottom-right', avatarUrl: 'Sofia_IA.png' });

⏰ Comportamento

SofiaWidget.init({ showAfterSeconds: 3, notificationDelay: 15000, exitIntentEnabled: true });

📊 Analytics

SofiaWidget.init({ analytics: true, // Integra com Google Analytics // e Facebook Pixel automaticamente });

📱 Mobile

SofiaWidget.init({ mobileOptimized: true, // Interface adaptada // para touch e mobile });

🎯 Presets Prontos

Configurações predefinidas para diferentes necessidades

🏛️ Estoico (Recomendado)

SofiaWidget.init( SofiaWidget.presets.estoic );

Cores e mensagem otimizadas para filosofia estoica

💼 Profissional

SofiaWidget.init( SofiaWidget.presets.professional );

Visual corporativo e comportamento discreto

⚡ Energético

SofiaWidget.init( SofiaWidget.presets.energetic );

Cores vibrantes e interação rápida

🎨 Minimalista

SofiaWidget.init( SofiaWidget.presets.minimalist );

Design clean e sutil

🎮 Controle Programático

Abra/feche a Sofia via JavaScript em eventos específicos

📝 Formulário Rejeitado

// Quando usuario não preenche form document.getElementById('form').addEventListener('invalid', () => { SofiaWidget.open(); });

⏰ Tempo na Página

// Após 30 segundos setTimeout(() => { SofiaWidget.open(); }, 30000);

📊 Scroll Específico

// Ao chegar na seção de preços const precosSection = document.getElementById('precos'); const observer = new IntersectionObserver(() => { SofiaWidget.open(); }); observer.observe(precosSection);

🔗 Botão Personalizado

// Botão "Falar com Consultora" document.getElementById('falar-consultora').addEventListener('click', () => { SofiaWidget.open(); });
